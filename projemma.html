Scratch Pseudocoding

Scratch is an educational programming language that was developed at the MIT. Scripts are made by dragging and dropping blocks together similar to a jigsaw puzzle.

For this project I created a game for my son based on "The Wiggles" Australian based kids music TV show. I utilized variables and events to create a scoring and level system. See if you can get a high score!
